Introduction¶

이 프로젝트의 목적은 Binary Instrumentation 기법을 다양한 플랫폼과 아키텍처에서 사용할 수 있도록 하는데 있습니다.

Main Features:

  • File Format Support : ELF, PE, MachO 실행파일 포맷 지원.
  • Instrumentation : ARM, x86 아키텍처 명령어의 Instrumentation 지원.

Warning

현재 Zigzi 프로젝트는 개발 테스트 버전으로 오직 Windows의 실행파일 포맷인 Portable Executable만을 한정적으로 지원합니다.
